This module teaches SQL by showing the aspects of manipulation, queries, aggregate functions, and how to create multiple tables. Queries were the most impactful part of this module as queries are the most important aspect of SQL, as it is what helps you sort through a table and filter a specific data that you want to examine. They are also important to store and update database information such as INSERT and UPDATE commands, which is very important for back end development and needs to have an embedded database, in order to be able to filter via the SELECT command through the information and give the computer enough information in order to provide a suitable front end page for the user to be able to process. It will help me to improve my learning of computer science, because now I will be able to understand and use different flavours of SQL, like PostgreSQL, MySql T-Sql.
